# CivicPulse - AI-Powered Civic Intelligence Platform

## Overview

CivicPulse is an AI-powered civic issue reporting and management platform designed to bridge the communication gap between citizens and local authorities.

The platform enables citizens to report civic problems such as garbage accumulation, water leakage, damaged roads, streetlight failures, and public safety concerns by providing descriptions, evidence images, and GPS-based locations.

On the other hand, authorities receive a centralized dashboard where incidents can be monitored, prioritized, assigned to relevant departments, and tracked in real time.

Our goal is to make civic issue resolution faster, transparent, and data-driven.

---

## Problem Statement

Traditional civic complaint systems often suffer from:

- Delayed reporting and response
- Lack of proper evidence and location information
- Poor communication between citizens and authorities
- No prioritization mechanism for critical incidents
- Limited transparency in complaint resolution

These challenges lead to inefficient management of civic issues and reduced public trust.

---

## Our Solution

CivicPulse provides an intelligent platform that combines AI, real-time reporting, image evidence, and location intelligence to improve civic issue management.

The platform offers:

- Real-time civic issue reporting
- Evidence image uploads
- GPS-based incident locations
- AI-assisted issue classification
- Priority-based incident management
- Authority dashboard for monitoring and resolution
- Complaint report generation and tracking

## Repository Setup

Clone the repository:

```bash
git clone https://github.com/riturajsingh0001/CivicPulse.git
cd CivicPulse
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build project:

```bash
npm run build
```
